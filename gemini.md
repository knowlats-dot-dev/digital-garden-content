Gemini is a [[ai|AI]] application developed by Google

ทำไม Gemini เป็นตัวเลือกที่ดี จากพี่ทอย ที่สอน [[course-data-science-bootcamp-12|Course: Data Science Bootcamp #12]]
เพราะ

**มี Ecosystem ที่ Strong** 

- เราสามารถสั่งให้ Gemini ส่ง Gmail หรือเซฟข้อมูลผ่าน Drive ได้
- ใช้ร่วมกับ Google App ได้ 
	- Spreadsheet ใช้สูตร `=AI('...')` เพื่อให้ Gemini ช่วยหาคำตอบใน Sheet นั้นได้

Model มีให้เลือก:

- Flash = เหมาะกับงานเร็ว
- Pro = คิดก่อนตอบ
- Imagen = สร้างรูป มี Nano Banana สามารถตัดต่อรูปได้
- Veo = สร้างวิดีโอ

Gemini is multimodal คือสื่อสารได้หลายรูปแบบ สร้าง text รูป วิดีโอ

Standard Formats 

- Simple Text 
- Bullet Point 
- Table 
- Image 
- Video
- Markdown
- CSV 
- JSON/ XML
- Code Block

Feature ที่มี

- Deep Research = ศึกษาข้อมูลเชิงลึก ทำ Report, Webpage, Infographic, Audio Podcast (สองคนคุยกัน) ได้
- Canvas = สามารถสร้างหน้าเว็บไซต์ได้ 
- Schedule = ตั้งเวลาให้ Gemini ทำงานที่ต้องการตามเวลา
- Gem = สร้าง Workspace ที่สามารถส่ง Data และ Instruction เฉพาะได้
